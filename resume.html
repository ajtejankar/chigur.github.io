Projects

Graph Based Git GUI 2015

This is my personal project that I am working on currently. Git is the most popular version control system today, but a lot of its users complain about the steep learning curve. All the GUI clients for it focus more on abstracting it rather than showing the underlying operations performed by Git. Because of this the user never has a clear idea of the implications of his actions. I am proposing a GUI client where user sees the underlying data structures used by Git and manipulates them directly. Inspired from - playground.
